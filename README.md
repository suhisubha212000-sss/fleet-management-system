# Fleet Management System

## Project Overview
A Python-based Fleet Management System used to manage vehicle information and analyze fleet performance.

## Features
- Vehicle details management
- Fleet summary generation
- Trip summary analysis
- Distance tracking
- Visualization using charts

## Technologies Used
- Python
- Pandas
- Matplotlib

## Files Included
- fleet_management_system.py
- fleet_summary.csv
- trip_summary.csv
- total_km_by_vehicle.png

## Output
The project generates reports and graphical analysis for vehicle distances and trip data.
